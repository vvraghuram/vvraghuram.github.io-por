## ✨ Features

- 🟢 **Scroll-Driven Ball Animation** — A glowing green ball travels down the Journey timeline in real-time sync with scroll position. Cards slide in as the ball hits each milestone.
- ⌨️ **Typewriter Effect** — Cycles through roles: Full Stack Developer, Cloud Engineer, IoT Builder, etc.
- 🖱️ **Custom Cursor** — Glowing dot + trailing ring cursor with hover scale effects.
- 📱 **Fully Responsive** — Mobile hamburger nav and adaptive grid layouts.
- 🎨 **Dark Aesthetic** — Deep navy/dark background with neon green (`#00e5a0`) and cyan (`#00b5ff`) accents.
- ⚡ **Parallax Glows** — Background blobs move on scroll for depth.
- 📬 **Contact Form** — Built-in contact section with form submission feedback.
- 🔝 **Scroll-to-Top Button** — Appears after scrolling 400px.

## 🟢 Journey Ball Animation — How It Works

The centrepiece of this portfolio is the **scroll-driven travelling ball**:

1. A **glowing green ball** sits on a vertical centre line
2. As you scroll through the Journey section, the ball **physically moves down** the line — synced 1:1 to your scroll position
3. A **green fill line** grows behind the ball as it travels
4. When the ball **reaches each milestone dot**, the card **slides in** from left or right
5. The ball does a **ripple/pulse pop** each time it hits a node
6. Scrolling **back up** reverses everything — cards un-light and ball retreats

## 🚀 Deploy to GitHub Pages

### Step 1 — Create Repository
1. Go to [github.com](https://github.com) → **New Repository**
2. Name it `portfolio` (or `<your-username>.github.io` for root domain)
3. Set to **Public**

### Step 2 — Upload Files
Upload all 4 files into the repository root:
- `index.html`
- `photo1.webp`
- `photo2.webp`
- `photo3.webp`

### Step 3 — Enable GitHub Pages
1. Go to **Settings** → **Pages**
2. Source: **Deploy from a branch**
3. Branch: `main` → `/root`
4. Click **Save**

### Step 4 — Visit Your Site
After ~60 seconds:
```
https://<your-username>.github.io/<repo-name>/
```
